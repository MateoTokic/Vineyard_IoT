# **Vineyard_IoT**

## **Mobile Application for Monitoring Wine Production Parameters and Processes**

**Vineyard_IoT** is an Android application developed as part of my **bacc.ing thesis**. The app is designed to assist winemakers in monitoring and managing key parameters during the wine production process. With features such as user authentication, real-time parameter monitoring, and note-taking, the application provides an efficient and user-friendly interface for winemakers.

The app is built using the **Kotlin programming language** and leverages **Firestore Database** for real-time data storage and synchronization.

---

## **Features**

- User authentication to securely log in.
- Real-time monitoring of parameters stored in the Firestore Database.
- Ability to view and record winemaker's notes.
- Intuitive and clean user interface for easy navigation.

---

## **Technologies Used**

- **Kotlin**: Programming language for Android development.
- **Firebase Firestore**: Cloud-based NoSQL database for real-time data storage and synchronization.
